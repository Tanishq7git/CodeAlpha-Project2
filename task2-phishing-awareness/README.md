# Task 2: Phishing Awareness Training

An interactive HTML training module covering how to recognize phishing emails and fake websites, common social engineering tactics, and best practices — finishing with a scored quiz.

## Features
- Annotated phishing email exhibit with clickable red-flag markers
- Annotated fake website / URL-spoofing exhibit
- Social engineering tactics dossier (urgency, authority impersonation, curiosity bait, reciprocity, brand impersonation, scarcity)
- Best-practice field guide checklist
- Composite real-world-style case studies
- 6-question interactive quiz with instant feedback and a final score

## Tech Used
HTML, CSS, JavaScript — no external dependencies, opens directly in any browser.

## How to Run
Open `phishing_awareness_training.html` in any browser (double-click the file, or right-click → Open With → your browser).
